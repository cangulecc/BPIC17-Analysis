# BPIC17-Analysis
Repository for the process mining analysis conducted for Bachelorpraktikum: Business Process Prediction, Simulation and Optimization. The objective was to analyze the BPI Challenge 2017 dataset, discover process models, and perform quality assessment to provide a reliable base model for simulation.
To ensure reproducibility, ensure the following Python packages are installed:
```bash
pip install pm4py pandas matplotlib numpy scikit-learn
```
# Dataset Setup
Due to size constraints, the raw event log is not included in this repository.

Download the official BPI Challenge 2017 XES dataset from Eindhoven University of Technology (TU/e): BPI Challenge 2017 Dataset.

Extract and place the file named BPI_Challenge_2017.xes directly into the root directory of this repository (the same folder as analysis.py).
# How to Run
1. Clone this repository to your local machine.
2. Install the required dependencies listed above
3. Execute the scripts within the analysis/ directory to reproduce the discovery results and quality metrics.
